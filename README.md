# Automation Task Bot

## 🚀 What it does
This project simulates an automation system that runs tasks at scheduled intervals and logs activity.

## 🧠 Features
- Runs automated tasks continuously
- Logs each task execution with timestamp
- Simulates real-world automation workflows

## 🛠 Tech Used
- Python

## 💡 Why I built this
To demonstrate automation skills and how repetitive tasks can be handled programmatically.

## ▶️ How to run
1. Install Python
2. Run:
   python bot.py

## 📁 Output
The bot logs all activity in logs.txt
